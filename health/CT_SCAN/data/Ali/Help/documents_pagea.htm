Documents page

The Documents page is displayed if the MI Disc View CD contains patient documentation associated with the open study. See also, Opening studies.

Illustration of the Documents page

The following table describes the Documents page components:

Component

Description 

1. Study Navigation toolbar

Consists of tabs that enable you to display items of a study on various study pages.

The Documents page is displayed by selecting the Documentation tab. For details, see Study Navigation toolbar.

2. Main toolbar

Enables you to:

  • Display the Study list (if available)
  • Launch the Advanced Viewer (if available)
  • Launch the online Help

For details, see Main toolbar.

3. Documents sidebar

Displays the patient name and Study Information folder. The Study Information folder contains a list of patient documentation associated with the open study. The type of documentation is identified by an icon and link. For details, see Viewing reports and Playing voice clips.

4. Document display area

Displays the selected patient document.

For details, see Viewing documentation.
