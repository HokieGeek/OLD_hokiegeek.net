Main toolbar

The main toolbar is located in the top right-hand corner of the study pages work area. The group of icons the toolbar contains may vary, depending on the contents of the study currently being viewed.

Illustration of the main toolbar

The following table describes the main toolbar icons:

Icon

Description

Advanced Viewer

Launch the Advanced Viewer to use the image manipulation tools when viewing images. The icon is available only on the Images page. For details, see Using the Advanced Viewer.

Note: The Advanced Viewer is an optional viewer and may not be available on your CD.

Study list

Display the Study list to open a study. For details, see Displaying the Study list.

Note: Not available on a CD that contains a single study.

Help

Launch the online Help. For details, see Finding help.
