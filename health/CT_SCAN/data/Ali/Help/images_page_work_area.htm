Images page work area

The Images page is displayed if the MI Disc View CD contains images of the open study. For details on viewing images in the Quick Viewer, see Viewing images.

Illustration of the Images page

The following table describes the Images page components:

Component

Description 

1. Study Navigation toolbar

Consists of tabs that enable you to display items of a study on the various study pages.

The Images page is displayed by selecting the Images tab. For details, see Study Navigation toolbar.

2. Main toolbar

Enables you to:

  • Display the Study list (if available)
  • Launch the Advanced Viewer (if available)
  • Launch the online Help

3. Study details

Provides the following information about the open study:

  • Patient name
  • Study date
  • Procedure type

4. Thumbnail Preview area

Displays the thumbnail images that represent the series of an open study with multiple series. It is located on the left-hand side of the Images page. For details, see Thumbnail Preview area.

5. Viewport

Area on the page where the image is displayed. For details, see Viewports and Viewing images in a series.
