Viewing reports

The reports associated with a study are listed on the Documents page. The method used to view the report depends on the format in which it was created.

If...

Then...

Created in ASCII or RTF report format

Viewed in HTML format, in the Web browser

Created in PDF format

Viewed in PDF format, in Adobe® Reader®

Note: The Adobe® Reader® can be downloaded from the web site at: www.adobe.com

To view a report:

  1. Click the Documents tab to display the Documents page, if it is not already displayed.

  2. In the Documents sidebar, click the report you want to view.

    Report icon

    Meaning

    Final report. A report that cannot be modified or resaved.

    Preliminary report. A report that may still require modification after further review.

    3. The report is displayed in the Documents display area, in HTML or PDF format.
