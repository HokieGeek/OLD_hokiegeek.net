Viewports

The viewport displays the images of a study. It also contains the navigation tools needed to page through the images.

Illustration of a viewport

The following table describes the viewport components.

Component

Description

1. Series Navigation bar

Note: The Series Navigation bar is displayed only for series that contain multiple images.

Bar used to move through a series. The bar consists of the following:

  • Navigation arrows. The arrows enable you to move through the series, viewing one image at a time.

  • Chits. Each chit represents an image in the series. The chits are highlighted as you page through a series.

For details on using the Series Navigation bar, see Viewing images in a series.

2. Image viewing area

Part of the screen that displays the actual images of a series.
