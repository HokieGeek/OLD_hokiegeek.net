Thumbnail Preview area

The Thumbnail Preview area provides an overview of the open study's images, as thumbnails. The thumbnails represent the first image of each series in the study. By selecting the appropriate thumbnail, you can choose which series to display in the viewport. For details, see Displaying series.

The Thumbnail Preview area is displayed only if the open study contains multiple series. It is located on the left-hand side of the Images page.

Illustration of the Thumbnail Preview area

The following table describes the Thumbnail Preview area components.

Component

Description 

1. Thumbnail

Provides a preview of the first image in each series. For details on using the thumbnail images to view series, see Displaying series.

2. Series information

Displays the following information about the series, below the thumbnail:

  • Series number
  • Number of images in the series
