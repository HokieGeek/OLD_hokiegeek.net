Printing images

You can print images in a study using the tools provided by the Web browser.

To print an image:

  1. Display the image you want to print.
  2. To print, do any of the following:
    • On the Web browser toolbar, click the Print icon.

    • On the Web browser menu bar, select Print from the File menu.
    • Press Ctrl + P.
    • Right-click the image or report, and then select Print from the menu.

    The Print dialog box is displayed.

  3. In the Select Printer box, select the appropriate printer.
  4. Click Print.
