Printing reports

You can print reports associated with a study.

To print a report:

  1. Display the report you want to print.
  2. In the Document Display area, do the following depending on the report format:

    If...

    Then...

    The report is displayed in HTML format
    • Click the Show Printer-Friendly Version link in the Document Display area.

    A print preview of the report is displayed in a separate window.

    The report is displayed in PDF format
    • Click the Print icon on the Adobe® Reader® toolbar.

    3. The Print dialog box is displayed.

  3. In the Print dialog box, do the following if required:
    • Select the printer you want to use to print the report
    • Specify the portion of the document you want to print
    • Specify the number of copies you want to print
  4. Click Print. If the report is in PDF format, click OK.
