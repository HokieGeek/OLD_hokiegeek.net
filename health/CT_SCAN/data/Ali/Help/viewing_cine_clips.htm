Viewing cine clips

The cine clips associated with a patient's study are listed on the Cine Clips page. Cine clips can be stored on the MI Disc View CD in any of the formats described in the following table. The formats are specified when the CD is created.

Format

Description 

Apple QuickTime™ (MOV)

A video format used in exporting cine clips to the MI Disc View CD. Cine clips exported in Apple QuickTime™ format are viewed using the Apple QuickTime™ Player.

Video for Windows® with MJPEG Compression (AVI)

A format developed by Windows® for storing video and audio information. Video for Windows® supports several data compression techniques, including MJPEG compression.

Cine clips stored in Video for Windows® format are viewed using the Windows Media® Player.

Note: To view cine clips in this format, you need to have the Pegasus MJPEG codec installed. The Pegasus PICVideo MJPEG codec for Windows® can be purchased or evaluated from the Pagasus web site at: http://www.jpeg.com/

Video for Windows® with Video 1 Compression (AVI)

A format developed by Windows® for storing video and audio information. Video for Windows® supports several data compression techniques, including Video 1 compression.

Cine clips exported in Video for Windows® format are viewed using the Windows Media® Player.

To view a cine clip:

  1. Click the Cine Clips tab to display the Cine Clips page, if it is not already displayed.

  2. Click the cine clip you want to view.

    3. The cine clip is displayed using the default media player, according to the file format. For details, see the table in this section.
