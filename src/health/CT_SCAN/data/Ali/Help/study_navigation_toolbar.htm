Study Navigation toolbar

The Study Navigation toolbar consists of a number of tabs that enable you to display the study pages available for the open study. Each tab represents a study page and each page displays specific items associated with the open study. For example, if an open study contains patient documentation and images, the Study Navigation toolbar consists of two tabs, the Documents tab and the Images tab.

The Study Navigation toolbar is located in the top left-hand corner of the study pages work area.

Illustration of the Study Navigation toolbar

The following table describes each tab on the Study Navigation toolbar, according to the study page it represents.

Tab

Description

1. Documents tab

Displayed if the open study contains patient documentation.

  • Click to view the Documents page.

For details, see Viewing documentation.

2. Images tab

Displayed if the open study contains images.

  • Click to view the Images page.

For details, see Viewing images.

3. Cine Clips tab

Displayed if the open study contains cine clips.

  • Click to view the Cine Clips page.

For details, see Viewing cine clips.
