About the study pages

Patient documentation and images are displayed on study pages. Each study page displays specific items, such as reports or images, associated with the open study. Only study pages that contain items are displayed. For example, if a study contains patient documentation but no images or cine clips, only the Documents page will be displayed.

The study pages are accessed from the Study Navigation toolbar. The following table describes each page and  the items it displays.

Page

Description 

Documents page

Lists the Reports and Voice clips associated with an open study.

For details, see Viewing documentation.

Images page

Displays the images in an open study. For details, see Viewing images.

Cine Clips page

Displays the cine clips in an open study. For details, see Viewing cine clips.
