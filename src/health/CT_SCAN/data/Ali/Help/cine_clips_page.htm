Cine Clips page

The Cine Clips page displays a list of the cine clips of an open study. For details on viewing cine clips, see Viewing cine clips.

To view cine clips, you need to have the following software installed on your computer:

If...

Then...

Viewing cine clips in Apple QuickTime™ (MOV) format

Install the Apple QuickTime™ Player.

Note: The Apple QuickTime™ Player can be downloaded free of charge from the web site at: http://www.apple.com/quicktime/

Viewing cine clips in Audio Video Interleave (AVI) format

Install the Windows Media® Player.

Note: The Windows Media® Player can be downloaded free of charge from the web site at: http://www.microsoft.com/windows/windowsmedia/en/download/default.asp.

Illustration of the Cine Clips page

The following table describes the Cine Clips page components:

Component

Description 

1. Study Navigation toolbar

Consists of tabs that enable you to display items of a study on the various study pages.

The Cine Clips page is displayed by selecting the Cine Clips tab. For details, see Study Navigation toolbar.

2. Main toolbar

Enables you to:

  • Display the Study list (if available)
  • Launch the Advanced Viewer (if available)
  • Launch the online Help

3. List of cine clips

Provides the following information about the listed cine clips:

  • Patient name
  • Modality
  • Link to play the cine clip, indicating the cine clip format. Cine clips may be viewed in either AVI or MOV format.
